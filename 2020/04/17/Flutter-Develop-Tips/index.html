Android打包

1.vscode 跑真机 Xcode报错dyld: Library not loaded: @rpath/App.framework/App

由于之前把APP.Framework 移除了，使用真机重新编译，后来手动导入时候没有加载。

把Embed状态改成Embed & Sign即可。

2.vscode 调试快捷键：

3.使用第三方库报错Exception: MissingPluginException(No implementation found for method

这是应为iOS没有被pod , 关闭as或者vs code再打开会重新编译，编译器会读取后自动pod。也可以手动pod install 一下也行。

4.Android权限

5.iOS加载启动页的资源放到跟目录下，不要放到Assets.xcassets，这样子才会打开就加载成功，不然会白屏一下子。

6.{}代表可选参数，可以不传类似swift有了默认值的参数

7.[!] No `Podfile' found in the project directory.

报这个错误如果Podfile是存在的，可以检查下目录路径是不是包含多个中文文件夹，换个目录就可以解决了
